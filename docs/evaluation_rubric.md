# SA_SLM 평가 체크리스트 및 루브릭

이 문서는 SA_SLM이 생성한 결과물의 품질을 평가하기 위한 기준을 정의합니다.

---

## 1. 로드맵 평가 체크리스트

### 1.1 구조적 완성도

| 항목 | 체크 | 설명 |
|------|------|------|
| 학기별 구조 | ☐ | 학기/학년별로 활동이 구분되어 있는가? |
| 활동 유형 다양성 | ☐ | 교과탐구, 동아리, 프로젝트, 독서, 봉사 등 다양한 유형이 포함되어 있는가? |
| 활동 설명 구체성 | ☐ | 각 활동에 대한 설명이 구체적인가? |
| 활동 간 연결성 | ☐ | 활동들이 서로 연결되어 성장 스토리를 형성하는가? |

### 1.2 전공 연결성

| 항목 | 체크 | 설명 |
|------|------|------|
| 전공 키워드 포함 | ☐ | 희망 전공 관련 키워드가 적절히 포함되어 있는가? |
| 전공 깊이 | ☐ | 학년이 올라갈수록 전공 관련 활동이 심화되는가? |
| 전공 지식 연결 | ☐ | 각 활동이 어떤 전공 지식과 연결되는지 명시되어 있는가? |

### 1.3 사회·윤리적 관점

| 항목 | 체크 | 설명 |
|------|------|------|
| 사회 이슈 연결 | ☐ | 사회 관심 키워드가 활동과 연결되어 있는가? |
| 윤리적 고려 | ☐ | 기술/경영의 윤리적 측면이 언급되어 있는가? |
| 가치관 반영 | ☐ | 학생의 가치관이 활동 선택에 반영되어 있는가? |

### 1.4 차별화

| 항목 | 체크 | 설명 |
|------|------|------|
| 개별성 | ☐ | 학생 고유의 관심사와 특성이 반영되어 있는가? |
| 일반적 활동 회피 | ☐ | "모든 학생이 할 법한" 일반적인 활동만으로 구성되지 않았는가? |
| 창의적 접근 | ☐ | 전공과 사회 이슈를 연결하는 창의적 활동이 포함되어 있는가? |

## 2. 세특 문장 평가 체크리스트

### 2.1 형식적 요건

| 항목 | 체크 | 설명 |
|------|------|------|
| 글자 수 | ☐ | NEIS 제한(보통 500자) 이내인가? |
| 문장 완결성 | ☐ | 문장이 완결되어 있고 끊기지 않았는가? |
| 어법/맞춤법 | ☐ | 맞춤법과 문법이 정확한가? |

### 2.2 내용적 요건

| 항목 | 체크 | 설명 |
|------|------|------|
| 활동 과정 기술 | ☐ | 구체적인 활동 과정이 드러나는가? |
| 전공 연결성 | ☐ | 희망 전공과의 연결이 자연스럽게 드러나는가? |
| 자기주도성 | ☐ | 학생의 주도적인 역할이 드러나는가? |
| 성찰/깨달음 | ☐ | 활동을 통한 성찰이나 배움이 표현되어 있는가? |

### 2.3 할루시네이션·과장 방지 (FR-4)

| 항목 | 체크 | 설명 |
|------|------|------|
| 입력 기반 | ☐ | 입력된 활동 내용만을 기반으로 작성되었는가? |
| 수상/성과 추가 없음 | ☐ | 입력에 없는 수상, 등수, 대외 성과가 추가되지 않았는가? |
| 과장 표현 없음 | ☐ | "탁월한", "최고의", "놀라운" 등 과장된 표현이 없는가? |
| 사실 확인 가능 | ☐ | 문장의 내용이 활동 기록과 대조하여 확인 가능한가? |

## 3. 역량 평가 루브릭

### 3.1 학업역량 (5점 척도)

| 점수 | 설명 |
|------|------|
| 5 | 깊이 있는 지적 탐구, 자기주도적 학습, 비판적 사고가 탁월하게 드러남 |
| 4 | 탐구심과 학습 깊이가 충분히 드러남 |
| 3 | 기본적인 학습 태도와 탐구가 드러나나 심화 필요 |
| 2 | 학업 관련 내용이 피상적으로만 언급됨 |
| 1 | 학업역량 관련 내용이 거의 없음 |

**평가 키워드**: 탐구, 분석, 연구, 조사, 학습, 이해, 심화, 비판적, 논리적

### 3.2 진로역량 (5점 척도)

| 점수 | 설명 |
|------|------|
| 5 | 명확한 전공 연결, 체계적 진로 탐색, 전문성 발전 가능성이 탁월함 |
| 4 | 전공 관련성과 진로 탐색 노력이 충분히 드러남 |
| 3 | 진로 관련 내용이 있으나 연결이 약함 |
| 2 | 진로와의 연결이 피상적임 |
| 1 | 진로역량 관련 내용이 거의 없음 |

**평가 키워드**: 전공, 진로, 직업, 분야, 관심, 꿈, 목표, 계획, 발전

### 3.3 공동체역량 (5점 척도)

| 점수 | 설명 |
|------|------|
| 5 | 뛰어난 협업, 리더십, 사회적 기여가 구체적으로 드러남 |
| 4 | 협업과 사회적 역할이 충분히 드러남 |
| 3 | 공동체 활동이 있으나 역할이 불명확함 |
| 2 | 공동체 관련 언급이 피상적임 |
| 1 | 공동체역량 관련 내용이 거의 없음 |

**평가 키워드**: 협업, 협력, 팀, 리더, 봉사, 나눔, 소통, 배려, 공동체

## 4. 융합성 평가 기준

### 4.1 융합 수준 정의

| 수준 | 설명 | 예시 |
|------|------|------|
| 높음 | 전공 + 사회 + 산업/윤리가 자연스럽게 통합됨 | "AI 채용 시스템의 알고리즘 편향을 분석하고, 공정한 알고리즘 설계 방안을 제안" |
| 보통 | 2개 관점이 연결됨 | "머신러닝 프로젝트를 수행하며 데이터 프라이버시 이슈를 고려" |
| 낮음 | 1개 관점만 언급됨 | "파이썬으로 프로그램을 작성" |
| 없음 | 융합적 관점이 없음 | 단순 활동 나열 |

### 4.2 융합 관점 체크리스트

| 관점 | 체크 | 관련 키워드 |
|------|------|------------|
| 전공 연결 | ☐ | 전공, 분야, 학문, 이론, 개념, 원리 |
| 사회·윤리 연결 | ☐ | 사회, 윤리, 문제, 이슈, 영향, 책임 |
| 산업·실무 연결 | ☐ | 산업, 기업, 현장, 실무, 적용, 실제 |

## 5. 종합 평가 가이드

### 5.1 최종 검토 체크리스트

```
[ ] 1. 입력된 활동 내용과 대조하여 할루시네이션이 없는가?
[ ] 2. 과장되거나 검증 불가능한 표현이 없는가?
[ ] 3. 학업·진로·공동체 역량 중 최소 2개가 드러나는가?
[ ] 4. 전공과 사회적 관점이 연결되어 있는가?
[ ] 5. 글자 수 제한을 준수하는가?
[ ] 6. 학생의 개별성과 진정성이 드러나는가?
```

### 5.2 품질 등급

| 등급 | 기준 |
|------|------|
| A | 모든 체크리스트 항목 충족, 역량 평균 4점 이상 |
| B | 필수 항목 충족, 역량 평균 3점 이상 |
| C | 일부 항목 미흡, 보완 필요 |
| D | 다수 항목 미흡, 상당한 수정 필요 |

## 6. 사용 시 주의사항

1. **이 시스템은 보조 도구입니다.** 최종 생기부 확정은 반드시 교사가 수행해야 합니다.
2. **생성된 내용은 반드시 검토하세요.** AI가 생성한 내용에 오류나 부적절한 표현이 있을 수 있습니다.
3. **학생의 실제 경험을 기반으로 하세요.** 입력되지 않은 활동이나 성과를 추가하지 마세요.
4. **개인정보 보호에 유의하세요.** 학생 식별 정보가 포함되지 않도록 주의하세요.
